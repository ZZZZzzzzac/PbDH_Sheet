# Validator diagnostics

PackageIssue：

```text
level: fatal | error | warning | info | debug
code: stable framework code
text: objective description
path?: compatibility dot path
location?: { pointer, file?, line?, column? }
entities?: [{ kind, id?, index? }]
evidence?: [{ label, value }]
```

`fatal` 表示读取/解析失败；`error` 阻止 Sheet Tool 渲染；`warning` 允许渲染；`info/debug` 非阻塞。诊断由规则现场、Loader source map 与 parser 结果生成，不含 `suggestion`。
