# 领域图标动画网页制作规范

## 项目概述

为 11 个几何 SVG 图标制作纯白背景的领域选择按钮演示页。每个图标点击后触发对应的入场动画，显示领域中英文名称。所有图标最终整合到单页，支持键盘/按钮翻页。

## 动画设计原则

1. **无循环或有限循环**：循环动画统一播放 2 次后停止（`animation-iteration-count: 2`），不用 `infinite`。
2. **出场顺序**：每个领域的部件按照其语义顺序依次出现，先中心后外围、先主体后装饰。
3. **缓动函数**：刚猛动作用 `cubic-bezier(0.05,0.90,0.25,1)`，轻柔动作用 `ease-out`，弹入用 `cubic-bezier(0.34,1.56,0.64,1)`。
4. **CSS transform 与 SVG transform 冲突**：如果 SVG 元素上已有 `transform` 属性（如 `<rect transform="rotate(-45)">`），必须将其移到外层 `<g>` 包裹，否则 CSS 动画的 `transform` 会覆盖 SVG 原生 transform。
5. **缝隙修复**：相邻多边形共用顶点时，浏览器抗锯齿会产生可见细缝。将其中一个多边形的共用边偏移 0.2px 使其微量交叠即可消除。

## SVG 部件拆分规则

读取原始 SVG 文件后，按以下步骤处理：

1. 去掉 `<?xml>` 声明、`<defs><style>` 等无关标签
2. 保留所有 `<polygon>`、`<path>`、`<polyline>`、`<rect>` 元素
3. 每个元素添加 `class="ak-part ak-[部件名]"`，移除原有的 `class="cls-1"`
4. 如果元素上有 `transform` 属性，将其移到外层 `<g transform="...">`
5. 根据领域特性命名部件，按动画顺序排列

## 整合页规范 (all-domains.html)

### 布局结构

整合页包含左右翻页箭头、11 个页面、页码指示器。初始只显示黑色按钮，点击后触发动画。

关键 CSS 变量：
```css
.name-wrap 定位: left:calc(50% + 12px); top:calc(50% - 46px)
按钮激活后左移: transform:translateX(-80px)
激活类名: .page.activated
每个领域的动画选择器: .page.{id}.activated .ak-xxx
```

### JS 数据格式

```javascript
const icons = [
  {id:'yuanjian', cn:'远见', en:'FORESIGHT', vb:'0 0 50.44 43.47', parts:[
    {c:'ak-left-wing', d:'...', t:'polygon', o:'transform-origin:...'},  // o 为可选内联样式
    // ...更多部件
  ]},
  // ...更多图标
];
```

每个 part 对象：
- `c`: class 名（如 `ak-left-wing`）
- `d`: 坐标数据（polygon 的 points 或 path 的 d）
- `t`: 元素类型（`polygon` / `path` / `polyline` / `rect`）
- `tx`: 可选，SVG transform（如 `rotate(120)`），会自动包裹在 `<g>` 中
- `o`: 可选，内联 style 属性

### 特殊处理

- **奇迹光环**：在 JS 中额外创建一条相同 path，设 `fill="none" stroke="#fff"`，class 为 `ak-ring`
- **工业拧动**：在 JS 中将所有 part 包裹在 `<g class="ak-hex-group">` 中
- **中文逐字**：`icon.cn.split('').map(c=>'<span class="char">'+c+'</span>').join('')`

### 功能特性

- 键盘 ← → 翻页
- 翻页时重置页面状态（移除 activated class）
- 背景大轮廓：每个页面的全屏 SVG，opacity 0.045 描边，点击后 0.8s 延迟淡入
- 按钮激活后 pointer-events:none，翻页重置

## 常见错误和修复

| 问题 | 原因 | 修复 |
|------|------|------|
| CSS transform 覆盖 SVG transform | 元素上的 transform 属性被 CSS animation 的 transform 替换 | 将 SVG transform 移到外层 `<g>` |
| 中文乱码 | PowerShell 写文件时编码错误 | 用 `[System.Text.UTF8Encoding]::new($false)` + `WriteAllBytes` |
| 循环动画停不下来 | 用了 `infinite` | 改为 `2`（播放两次） |
| 相邻多边形有细缝 | 浏览器抗锯齿 | 偏移 0.2px 让相邻边微量交叠 |
| bottom 定位上移反向下移 | CSS bottom 值越小越靠下 | 改用 `top` 定位避免混淆 |
| 编组导致闪现 | 子元素 opacity 覆盖和动画延迟不同步 | 不编组，独立动画 |
