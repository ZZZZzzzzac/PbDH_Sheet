# 近卫

\[近卫职业简介\]

|【初始生命点】<br>7|【初始闪避值】<br>10|【初始主选领域】<br>攻坚|
|---|---|---|



### **【希望特性】**

短兵相接：在一个敌人对你进行了一次攻击掷骰后，你可以花费 3 希望点对攻击范围内的该敌人进行一次攻击反应掷骰。



### **【职业特性】**

狭路相逢：每次休息一次，标记 1 压力点，在你的一次攻击掷骰中孤立你和其中一个目标。

*\*孤立：只能成为其他角色攻击掷骰、恐惧动作或反应动作的目标。*
