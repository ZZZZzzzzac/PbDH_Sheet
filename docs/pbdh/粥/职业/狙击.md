# 狙击

\[狙击职业简介\]

|【初始生命点】<br>5|【初始闪避值】<br>10|【初始主选领域】<br>精准|
|---|---|---|



### **【希望特性】**

动态瞄准：花费 3 希望点，锁定你攻击范围内的一个敌人，你能持续地确认其所在的位置且你对其进行攻击时熟练值获得\+1加值，直至其消失在你的视野中、离开你的攻击范围一段时间后或切换场景时。



### **【职业特性】**

找准时机：每次休息一次，当你标记压力点时，你可以将宝库中一张回想费用不大于此次标记压力点数目的领域卡加入配置中。

*\*不视为通过回想加入配置中。*

# 〖重射手〗

推荐搭配领域：**【迅攻】**、【攻坚】

|---|
|## 正式干员<br>武器原型：复合弩 中距离/双手/物理<br>武器伤害骰：d10\+3<br>### **子职特性提升**<br>锐矢洞穿\+：你使用复合弩的攻击可以穿透可摧毁物件（如较薄的砖瓦墙体）和大部分生物躯体，且射速难以被肉眼捕捉。若你攻击的目标在你的近距离内，你可以标记 3 压力点，在此次攻击中使你的熟练值获得你当前位阶数目\+1的加值。|
|## 资深干员<br>武器原型：复合弩 中距离/双手/物理<br>武器伤害骰：d10\+6<br>### **职业特性提升**<br>找准时机\-重射手：每次休息一次，当你标记压力点或对一个目标造成伤害时，你可以将宝库中一张回想费用不大于此次标记压力点或目标标记生命点数目的领域卡加入配置中。<br>### **子职特性二次提升**<br>锐矢洞穿\+\+：你使用复合弩的攻击可以穿透可摧毁物件（如较薄的砖瓦墙体）和绝大部分生物躯体，且射速难以被肉眼捕捉。若你攻击的目标在你的近距离内，你可以标记 2 压力点，在此次攻击中使你的熟练值获得你当前位阶数目\+1的加值。|
|## 精英干员<br>武器原型：复合弩 中距离/双手/物理<br>武器伤害骰调整值\+3<br>### **【X模组】**<br>#### 希望特性提升<br>动态瞄准\-重射手：花费 3 希望点，锁定你攻击范围内的一个敌人，你能持续地确认其所在的位置且你对其进行攻击时熟练值获得\+2加值，直至其消失在你的视野中、离开你的攻击范围一段时间后或切换场景时。<br>---<br>### **【Y模组】**<br>#### 追加第二职业特性<br>钢射逐风：尝试中断你攻击或改变你攻击目标的敌人会成为你此次攻击的额外目标（而此次攻击仍会包含你原本指定的目标）。若你攻击的目标在你的近距离内，此次攻击造成的伤害无视抗性和免疫。|

# 〖神射手〗

推荐搭配领域：**【奇迹】**、【攻坚】

|## 预备干员<br>武器原型：狙击枪或狙击弩 远距离/双手/物理<br>武器伤害骰：d12<br>### **【子职特性】**<br>定点狙击：在你进行一次移动后，你无法在当前聚焦中进行攻击掷骰。当你进行攻击掷骰时，若你在上一次聚焦中未进行过移动，你的狙击武器攻击范围扩大至当前场景，且你可以标记 1 压力点，在此次攻击掷骰中获得\+1加值。|

|## 正式干员<br>武器原型：狙击枪或狙击弩 远距离/双手/物理<br>武器伤害骰：d12\+3<br>### **子职特性提升**<br>定点狙击\+：在你进行一次移动后，你在当前聚焦中进行攻击掷骰须额外标记 3 压力点。当你进行攻击掷骰时，若你在上一次聚焦中未进行过移动，你的狙击武器攻击范围扩大至当前场景，且你可以标记 1\-2 压力点，在此次攻击掷骰中获得等量加值。|

|## 精英干员<br>武器原型：狙击枪或狙击弩 远距离/双手/物理<br>武器伤害骰调整值\+3<br>### **【X模组】**<br>#### 希望特性提升<br>动态瞄准\-神射手：花费 3 希望点，锁定你攻击范围内的一个敌人，你能持续地确认其所在的位置且你对其进行攻击时熟练值获得\+1加值，其始终视为处于你的攻击范围内，直至其消失在你的视野中、你锁定另一个敌人或切换场景时。<br>---<br>### **【Y模组】**<br>#### 追加第二职业特性<br>耐压之心：每当你对中远距离交界处以外的敌人进行一次成功的攻击后，你可以清除 1 压力点。每场景一次，当团队移交聚焦时，若你在上一次聚焦和此次聚焦中均未进行过移动，你可以更好地放松心态，清除 2 压力点。|
