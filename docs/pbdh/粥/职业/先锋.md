\[先锋职业简介\]

|【初始生命点】<br>6|【初始闪避值】<br>12|【初始主选领域】<br>迅攻|
|---|---|---|



### **【希望特性】**

临危寻变：花费 3 希望点，你立即进行1至2次回想费用恒定为 0 的回想。



### **【职业特性】**

一鼓作气：每次休息一次，标记 1 压力点，你可以重掷一次攻击掷骰。若此次重掷取得关键成功，你可以在攻击完成后进行一次近距离移动。

# 〖战术家〗

推荐搭配领域：**【工业】**、【奇迹】

|## 预备干员<br>武器原型：战术召唤物 远距离/单手/物理<br>武器伤害骰：d6<br>### **【子职特性】**<br>战术协同：描述你的武器原型是怎样的（生物或造物），它会与你一同进行攻击和移动。每场游戏开始时，将一枚d10召唤骰放置在你配置中的一张领域卡上，将其作为你的战术召唤物使用，召唤骰初始数值为1。战术召唤物具有对应领域卡特性（使用频率与你共同计算），其的伤害阈值与你保持一致。其标记生命点时，改为使召唤骰数值增加等数目。当召唤骰数值大于对应领域卡等级时，战术召唤物失去战斗能力，直至召唤骰数值为1。当你完成一次短休/长休后，使召唤骰数值减少5/减少10（召唤骰数值至少为1）。|
|---|
|## 正式干员<br>武器原型：战术召唤物 远距离/单手/物理<br>武器伤害骰：d6\+3<br>### **子职特性追加**<br>战术协同\+：当战术召唤物近距离内的自愿盟友在一次攻击掷骰中获得成功结果时，你可以标记 2 压力点，让战术召唤物协同盟友进行攻击，将使用你熟练值一半的武器伤害合并计入盟友的伤害掷骰中。|
|## 资深干员<br>武器原型：战术召唤物 远距离/单手/物理<br>武器伤害骰：d6\+6<br>### **职业特性提升**<br>一鼓作气\-战术家：每次休息一次，标记 1 压力点或使战术召唤物的召唤骰数值增加1，你可以重掷一次攻击掷骰。若此次重掷取得关键成功，你可以在攻击完成后进行一次近距离移动。<br>### **子职特性二次追加**<br>战术协同\+\+：当你在一次动作掷骰中获得希望成功/关键成功时，你可以使召唤骰数值减少1/减少3（召唤骰数值至少为1，下同）。你的盟友获得关键成功时，你也可以使召唤骰数值减少3。|
|## 精英干员<br>武器原型：战术召唤物 远距离/单手/物理<br>武器伤害调整值\+3<br>### **【X模组】**<br>#### 希望特性提升<br>临危寻变\-战术家：花费 3 希望点，你立即进行1至3次回想费用恒定为 0 的回想。<br>---<br>### **【Y模组】**<br>#### 追加第二职业特性<br>最佳拍档：当战术召唤物的召唤骰数值不大于 3 时，其的伤害骰提升至d8而非d6。你使用战术召唤物进行攻击时，若其的召唤骰数值为 1，你的攻击掷骰和熟练值均获得\+2加值。|
