## DaggerHeart 角色卡工具功能

### 角色与存档

- 支持多个角色存档。
- 支持新建、切换、重命名、复制、删除角色存档。
- 支持点击当前存档名快速重命名。
- 支持自动保存当前角色数据到本地浏览器存储。
- 支持从 JSON 创建新存档。
- 支持从 HTML 创建新存档。
- 支持旧数据导入时补齐缺失字段。

### 角色卡编辑

- 支持填写角色基础信息：姓名、等级、头像。
- 支持选择职业、子职业、种族、社群。
- 选择职业后可自动带入职业相关初始信息。
- 支持编辑六项属性：敏捷、力量、灵巧、本能、风度、知识。
- 支持标记施法属性。
- 支持编辑闪避、生命槽、压力槽、希望、护甲槽。
- 支持编辑熟练度。
- 支持编辑伤害阈值、护甲值、护甲加值。
- 支持编辑经历及经历数值。
- 支持编辑物品栏、金币、备用武器。
- 支持编辑职业特性文本。
- 支持上传并裁剪角色图片。

### 装备选择

- 支持从武器库选择主武器、副武器、备用武器。
- 选择武器后自动填写名称、类型/负荷/范围、伤害、特性。
- 支持自定义武器数据。
- 支持从护甲库选择护甲。
- 选择护甲后自动填写护甲名称、护甲值、阈值、特性。

### 卡牌与卡组

- 支持内置卡牌库。
- 支持自定义卡牌库。
- 支持卡牌类型：职业、种族、社群、子职业、领域、变体。
- 支持角色聚焦卡组。
- 支持角色库存卡组。
- 支持在聚焦卡组和库存卡组之间移动卡牌。
- 支持删除卡牌。
- 支持添加卡牌到下一个空位。
- 聚焦卡组前 5 个特殊卡位与职业、子职业、种族、社群选择同步。
- 支持按卡包筛选卡牌。
- 支持按类别筛选卡牌。
- 支持按等级筛选卡牌。
- 支持按名称、描述、显示字段搜索卡牌。
- 支持无限滚动浏览卡牌。
- 支持卡牌详情预览。
- 支持在选卡时快速创建一张自定义变体卡。

### 页面与显示

- 默认有第一页和第二页角色卡。
- 可选显示游侠伙伴页。
- 可选显示主板扩展页。
- 可选显示冒险笔记页。
- 页面按 A4 尺寸组织，支持打印。
- 支持页面标签切换。
- 支持页面显示管理。
- 支持单页模式。
- 支持桌面端双页模式。
- 支持左右箭头切换页面。
- 支持数字键跳转页面。
- 支持 `Ctrl + 数字` 切换角色存档。
- 支持 `Esc` 退出导出预览。
- 支持图片模式和文字模式切换。

### 第二页与升级

- 支持填写角色描述、背景、外貌、动机等文本。
- 支持显示和编辑卡组。
- 支持按等级段记录升级选项。
- 支持勾选升级项。
- 支持升级项影响生命槽上限、压力槽上限、熟练度、闪避值、经历数值、属性值。
- 支持撤销部分升级并尝试回滚对应数值。
- 支持从升级项选择领域卡。
- 支持从升级项选择子职业卡。

### 可选角色附页

- 游侠伙伴页支持：伙伴名称、伙伴图片、伙伴描述、伙伴经历、伙伴经历数值、攻击方式、伤害骰、闪避、压力、压力上限、训练选项。
- 冒险笔记页支持：角色立绘、角色简介、玩家信息、人物小传、大事记、冒险履历。
- 主板扩展页支持扩展型装备/武装记录，包括武器信息、升级槽、升级项、材料/碎片记录、电子货币等。

### 建卡指引

- 支持按步骤引导建卡。
- 引导内容包括：选择职业、选择子职业、选择种族、选择社群、分配属性、记录等级/闪避/生命、记录压力与希望、选择初始武器、选择初始护甲、添加初始物品、填写背景与关系、选择能力卡牌、添加经历或特质、最终检查。
- 部分步骤会根据当前角色选择显示对应提示。
- 部分步骤会检查当前表单是否满足继续条件。

### 导入导出与打印

- 支持打开导出预览。
- 支持导出 PDF。
- 支持导出 HTML。
- 支持导出 JSON。
- 支持等待图片加载后再导出。
- 支持导出时按角色信息生成页面标题。
- 支持从 HTML 导入角色。
- 支持从 JSON 导入角色。
- 支持导出到海豹骰子。
- 支持生成 `.st` 属性命令。
- 支持复制骰子导出文本。
- 支持下载配套骰子脚本。
- 支持查看骰子使用指南。

### 卡包管理

- 支持导入自定义卡包。
- 支持导入格式：JSON、DHCB、ZIP。
- 支持多文件批量导入。
- 支持显示导入成功、失败、错误信息、重复 ID、导入卡牌数量、导入图片数量。
- 支持查看所有卡牌。
- 支持查看某个卡包内的卡牌。
- 支持启用或禁用卡包。
- 支持删除单个卡包。
- 支持清空所有自定义卡牌。
- 支持查看本地存储使用情况。
- 支持强制初始化所有本地数据。
- 支持查看卡包创作指南、AI 创作指南和示例 JSON。

### 卡包编辑器

- 支持新建卡包。
- 支持导入卡包。
- 支持导出卡包。
- 支持验证卡包。
- 支持查看关键字/预定义字段列表。
- 支持编辑卡包基础信息。
- 支持编辑职业卡。
- 支持编辑种族卡。
- 支持编辑社群卡。
- 支持编辑子职业卡。
- 支持编辑领域卡。
- 支持编辑变体卡。
- 支持卡牌预览。
- 支持卡牌列表快速跳转。
- 支持复制卡牌。
- 支持删除卡牌。
- 支持上传卡牌图片。
- 支持输入图片 URL。
- 支持导出包含图片的卡包。
- 支持管理预定义字段：职业、种族、社群、领域、变体类型。

### 悬浮笔记本

- 支持打开/关闭悬浮笔记本。
- 支持拖动笔记本位置。
- 支持最多 5 页笔记。
- 支持添加、删除、切换页面。
- 支持每页最多 10 行。
- 支持文本行。
- 支持计数器行。
- 支持骰子行。
- 支持行内容编辑。
- 支持删除行。
- 支持调整行顺序。
- 支持添加骰子、删除骰子、重投骰子、手动编辑骰子值。

## “我的车技如何？”玩家车卡器功能

### 角色档案

- 支持填写角色姓名。
- 支持选择角色原型。
- 支持填写面部选角。
- 支持填写角色气质。
- 选择原型后自动填充增益。
- 选择原型后自动填充缺陷。

### 玩家资源

- 支持选择燃料骰：`d12`、`d10`、`d8`、`d6`、`d4`、`OUT OF GAS`。
- 支持填写燃料点。

### 行事风格

- 支持从预设行事风格列表中选择 4 项。
- 4 项分别对应：专家 +3、擅长 +2、熟练 +1、生疏 -1。

### 纪念物与经历

- 支持填写 3 个纪念物。
- 每个纪念物支持填写一条对应经历。
- 页面说明经历可通过花费 1 燃料点获得 +2 加值。

### 座驾

- 支持选择车型。
- 选择车型后自动填充座驾骰。
- 选择车型后自动填充座驾特性。
- 支持记录座驾损伤点。
- 支持填写 3 个装备栏位。

### 文件与打印

- 支持保存车卡为 JSON。
- 支持读取 JSON 车卡。
- 支持打印或导出 PDF。

## “我的车技如何？”GM 工具功能

### 顶层模块

- 支持在 3 个模块之间切换：调查档案、生存指南、GM 仪表盘。

### 调查档案

- 提供司机资料。
- 提供司机“杀戮”规则：花费 1 恐惧点，`1d20 + 杀戮骰` 对抗玩家反应。
- 提供限速到杀戮骰的映射：1-5 为 d4，6-10 为 d6，11-15 为 d8，16-20 为 d10，21+ 为 d12。
- 提供司机行为速查：躲开那个司机、惊恐、路怒、极速、催眠、偏离。
- 提供站点类型速查：休息站点、维修站点、交通站点。
- 提供当地居民资料：店主、警长、机修工。
- 提供当地居民资源、服务、恩惠、威吓/任务信息。
- 提供搭车者资料：绅士、醉汉/醉妞、纨绔子弟。
- 提供搭车者出现条件、增益、缺陷、相关事件。

### 生存指南

- 提供游戏循环速查：逃离司机、在路上获取燃料点、在站点加油、击退司机、重复循环、进入最后冲刺。
- 提供限速规则：限速是动作掷骰目标数。
- 提供限速 30 或以上时游戏立即结束的规则。
- 提供追逐指示物规则：GM 获得恐惧点时同时获得 1 追逐指示物。
- 提供追逐指示物满 6 时的处理：司机出现、GM 获得玩家数量等量恐惧点、启动倒计时、清空追逐指示物。
- 提供路程行动规则。
- 支持 3 种路程行动说明：直面恐惧、建立联系、消磨时间。
- 提供聚焦轮换与驾驶员轮换规则。
- 提供站点休整规则：抵达站点时燃料骰降低一级。
- 提供燃油耗尽规则。
- 提供冲刺阶段触发条件。
- 提供冲刺阶段特殊规则：限速变为最终倒计时，倒计时归零则司机永久击败，此阶段 GM 获得 2 点恐惧点。
- 提供生存行动选项：花费全部燃料点、当前限速 +1、燃料骰降低一级。
- 提供死亡抉择选项：殊死一搏、自我牺牲。

### GM 仪表盘

- 支持查看并调整当前限速。
- 支持限速加 1 和减 1。
- 支持点击限速数值直接输入。
- 支持查看并调整 GM 恐惧点。
- 支持恐惧点获得和消耗。
- 支持点击恐惧点数值直接输入。
- 支持追逐指示物计数。
- 支持追逐指示物加 1、减 1、清空。
- 支持自定义追逐指示物上限。
- 追逐指示物达到上限时显示“司机找到你了”提示。
- 支持查看座驾耐久骰等级。
- 支持点击座驾耐久条设置或降低座驾等级。
- 支持记录座驾损伤点。
- 支持损伤点加 1、减 1。
- 支持点击损伤点数值直接输入。
- 支持一个可命名倒计时。
- 支持倒计时加 1、减 1。
- 支持编辑倒计时标题。
- 提供“杀戮”动作提示。
- 提供“惊恐”动作提示。
- 提供 GM 手册速查：限速对应行程数量、限速对应杀戮骰。
- 当前限速变化时，高亮对应的行程数量规则。

## 对本仓库的需求转化建议

### 第一阶段

- 做角色存档管理。
- 做可编辑角色卡。
- 做规则项选择器。
- 做 JSON 导入导出。
- 做 PDF/打印导出。
- 做 GM 基础计数器：限速、恐惧、追逐、倒计时、座驾耐久、损伤点。

### 第二阶段

- 做建卡引导。
- 做卡牌/能力/条目库。
- 做自定义卡包导入。
- 做卡组或能力槽管理。
- 做本地多存档迁移。

### 第三阶段

- 做卡包编辑器。
- 做悬浮笔记本。
- 做骰子导出。
- 做双页模式。
- 做可选扩展页。

## 明确不纳入首批需求

- 纯动画。
- 纯视觉主题。
- 做旧纸张效果。
- 仪表盘震动、闪烁、火焰、裂纹、血滴等彩蛋特效。
- 与系统玩法无关的装饰性布局。
