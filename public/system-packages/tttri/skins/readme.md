美术UI风格参考: https://ak.2heng.xin/ (游戏内UI)  https://ak.hypergryph.com/#index (官网) https://ak.hypergryph.com/archive/dynamicCompile

可以参考的UI设计skill: /design-taste-frontend /redesign-existing-projects /image-to-code (注意这些skill部分内容是面向通用网页设计的, 不适合我们这个TRPG人物卡)

人物卡内的布局可以随意更改, html/css可以重写, 可以增加修改html静态对象, 如标题/logo/分割线等

保持左人物卡右卡牌桌面的布局, 不得增删修改人物卡内的模块(modules.json), 但可以重新组合排布它们的位置\大小\样式, 不必保留当前的原有布局

最终需要保持人物卡为2页A4纸以内, 每页的内容不得溢出, 同时也不得太过压缩导致模块内的文字/标记看不清. 

目前已有一个黄黑配色的工业风皮肤, 你的风格选择要与其不同, 着重还原我给你美术UI风格参考的链接里的官网风格
